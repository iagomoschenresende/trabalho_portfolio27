PORTFÓLIO

É com grande satisfação dizer que possuo experiência com a linguagem SQL. Com o SQL (Structured Query Language), você é capaz de gerenciar bancos de dados relacionais, criar e modificar tabelas, inserir, atualizar e excluir dados, além de executar consultas complexas para obter informações específicas. Tornando-se uma habilidade valiosa no mercado de trabalho, especialmente em áreas de dados, análise de negócios e desenvolvimento de software.

Com satisfação, afirmo que eu tenho experiência com as linguagens HTML e CSS. O HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar conteúdo web, enquanto o CSS (Cascading Style Sheets) é usado para estilizar e formatar esse conteúdo, tornando-o visualmente atraente e intuitivo para os usuários. Minha experiência em HTML é evidente ao demonstrar habilidade em criar páginas web bem estruturadas, usando as tags apropriadas para definir a hierarquia do conteúdo e criar links e formulários interativos. Além disso, você também é capaz de usar o CSS para aplicar estilos visuais às páginas, como fontes, cores, layouts.

Minha experiência em IU/UX design é evidente em minha habilidade de criar wireframes, protótipos e designs finais usando ferramentas populares como balsamiq mockups, Figma e draw.io . Além disso, sou capaz de realizar testes de usabilidade e avaliações de produtos existentes, identificando áreas de melhoria e implementando soluções eficazes para melhorar a experiência do usuário.
